重置mysql的登录密码

mysql的用户名和密码保存在数据库mysql的user表里面；要重置密码，可以更新这个表的相关内容。
其中关键字段如下：

Field Type Null Key
Host char(60) NO PRI
User char(16) NO PRI
Password char(41) NO

具体方法：

  1. KILL掉系统里的MySQL进程

  2. 用以下命令启动MySQL，以不检查权限的方式启动

  3. 然后用空密码方式使用root用户登录 MySQL

  4. 修改root用户的密码

需要替换’新密码’为想要的密码；
flush privileges刷新权限；
重启。
